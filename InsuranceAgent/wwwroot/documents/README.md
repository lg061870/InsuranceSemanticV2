# Insurance Documents Directory

This directory contains PDF documents for processing and embedding into the vector database.

## Usage

1. Place PDF files in this directory
2. Use the Document Explorer interface to process and embed documents
3. Search across embedded documents using semantic search

## Supported File Types

- PDF files (.pdf)
- Other document formats may be added in the future

## Processing

Documents placed in this directory can be:
- Processed individually or in batches
- Embedded into vector database collections
- Searched using semantic similarity

## Access

This directory is served under the `/documents/` web path for easy access and management.
